🌐 Assetfinder: Subdomain Discovery Tool

Assetfinder is a powerful reconnaissance tool designed to find domains and subdomains potentially related to a given root domain. It's used extensively in OSINT, bug bounty hunting, and red teaming to expand attack surfaces quickly.

  • Goal: Discover related subdomains for a given target
  • Method: Passive enumeration using public APIs and sources
  • Use case: Bug bounty, pentesting, asset discovery

🔍 What is Assetfinder?

Assetfinder queries multiple online sources to gather domains, subdomains, and related infrastructure associated with your target. It’s written in Go, supports both CLI and scripting workflows, and uses public APIs like crt.sh, HackerTarget, VirusTotal, and more.

📦 Supported Data Sources

  • crt.sh: Certificate Transparency logs
  • certspotter: SSL certificate search
  • threatcrowd: Threat intel aggregator
  • hackerTarget: DNS-based subdomain discovery
  • VirusTotal: Malware database (needs VT_API_KEY)
  • Facebook: Domain detection via Graph API (needs FB_APP_ID and FB_APP_SECRET)
  • Spyse / FindSubDomains: Passive DNS search (needs SPYSE_API_TOKEN)
  • urlscan.io: Indexes scanned web pages and subdomains
  • dns.bufferover.run: Passive DNS records

⚙️ Installation

go install -v github.com/tomnomnom/assetfinder@latest

🚀 Usage

# Basic usage:
assetfinder example.com

# Only show subdomains:
assetfinder --subs-only example.com

📁 Sample Output

blog.example.com
cdn.example.com
mail.example.com
vpn.example.com

🔧 Advanced Features

  • Concurrency: Executes multiple API queries in parallel
  • Rate Limiting: Per-source delay to avoid bans
  • De-duplication: Avoids repeated results
  • Scriptable: Reads input from STDIN or CLI args

🧠 Summary for Beginners

If you're hunting for hidden subdomains during a bug bounty or recon phase, Assetfinder saves you time by aggregating results from trusted sources. Just give it a domain, and it gives you a list of potential assets to investigate—no DNS brute-force required.

Tip: Combine Assetfinder with tools like httpx and nmap for deeper recon.
